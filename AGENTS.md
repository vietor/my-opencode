# Global Restrictions

When generating or updating the `AGENTS.md` file, the principles described in this document must be followed.

## Language Restrictions
- All interactive content must be in Simplified Chinese.
- Generated help content should primarily be in Simplified Chinese.

## Code Generation Principles
- Do not generate code comments.
- Do not generate test code.
- Do not generate sample code.

## AGENTS.md File Generation Principles
- Do not generate deployment-related content.
- Do not generate testing-related content.
- Use terminology that is easily understood by artificial intelligence whenever possible.
